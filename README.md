# gestor de estudiantes

aplicacion python con interfaz en consola que permite manipular, organizar, crear e importar datos en una base de datos, ademas de la lectura en si de los mismos mencionados.
Incluye generacion de datos y creacion de una base de datos, ademas de la importacion de datos a esta misma ademas de un manejo mas avanzado mediante comandos mas complejos, permitiendo actualizar datos expecificos, borrar multiples datos a partir de un filtro, busqueda parciales a partir de caracteres similares.
